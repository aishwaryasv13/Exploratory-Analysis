Titanic Dataset — Exploratory Data Analysis (EDA)

Key Questions Explored

Did the port of embarkation affect survival chances?
How did passenger class (socioeconomic status) influence survival?
Did age play a role in who survived?
How did gender correlate with survival outcomes?

What's Inside

Data Cleaning — handled missing values in Age, Cabin, and Embarked columns using Available-Case Analysis
Feature Engineering — dropped irrelevant columns (Cabin, Ticket, Fare, Name) and mapped categorical keys to readable labels
Visualizations — bar charts, pie charts, and grouped analysis using Matplotlib and Seaborn
Survival Analysis — breakdown of survival rates by embarkation port, gender, age group, and passenger class

Tech Stack
Python Pandas NumPy Matplotlib Seaborn Jupyter Notebook
Dataset
Kaggle Titanic Dataset — 891 of 2,224 passengers with demographics and travel information.
